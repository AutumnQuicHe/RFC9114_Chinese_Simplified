对于响应，只定义了一个伪标头字段`:status`，它传递的是HTTP状态码；详见《[HTTP]()》的[第15章]()。
该伪标头字段 {{< req_level MUST >}}被包含在所有响应中；否则，该响应就是畸形的（详见[第4.1.2章]()）。

HTTP/3没有定义一种像HTTP/1.1的状态行那样传递版本或原因短语的方法。
HTTP/3响应隐式地使用了协议版本`3.0`。
